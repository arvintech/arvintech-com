## Client Page Template Structure

Each individual client page will contain:

1. **Hero Section**
   - Client logo (large)
   - Client name
   - Industry tag
   - Brief tagline

2. **About the Business**
   - Description of the business
   - What they do
   - Their mission/values

3. **Our Partnership**
   - What we do with them
   - Services provided
   - Technologies used

4. **How They Use ArvinTech**
   - Specific use cases
   - Results/outcomes
   - Testimonials (if available)

5. **Client Portal Access** *(Future Feature)*
   - Login for authenticated clients
   - Access to project specs
   - Communication hub
